# 12 — User Journey Map

## Title
FundiLink — Primary User Journey Map

## Purpose
Map the end-to-end learner journey through FundiLink to identify key touchpoints, emotions, and opportunities.

---

## Primary Journey: Thabo Applies to University via FundiLink

### Stage 1: Awareness
**Touchpoint:** Friend shares FundiLink link on WhatsApp / school counsellor mentions it
**Thabo's thought:** "What is this? Can it actually help me?"
**Emotion:** Curious, slightly sceptical
**FundiLink action:** Clear, simple landing page that explains the value in plain language
**Opportunity:** Make the value clear in 5 seconds. Don't use jargon.

---

### Stage 2: Registration
**Touchpoint:** Landing page → Create free profile
**Thabo's thought:** "This looks easy. Let me try."
**Emotion:** Hopeful, slightly anxious about privacy
**FundiLink action:** Simple registration form; email verification; POPIA consent in plain language; guardian consent if under 18
**Opportunity:** Reduce friction. Make consent understandable. Reassure about data safety.

---

### Stage 3: Profile Creation
**Touchpoint:** Onboarding flow — personal details, school info, grade
**Thabo's thought:** "What information do they need? This is a lot of boxes."
**Emotion:** Slightly overwhelmed
**FundiLink action:** Step-by-step guided profile creation; progress indicator; save-and-continue; required vs optional clearly marked
**Opportunity:** Break it into small steps. Show progress. Celebrate completion.

---

### Stage 4: Enter Academic Results
**Touchpoint:** Academic profile — enter NSC subjects and marks
**Thabo's thought:** "I know my marks. Let me put them in."
**Emotion:** Engaged, curious what APS will be
**FundiLink action:** Subject selector (from approved NSC subject list); mark entry; instant feedback
**Opportunity:** Make mark entry feel like a quiz, not a form.

---

### Stage 5: APS Calculation
**Touchpoint:** Profile → APS score displayed
**Thabo's thought:** "My APS is 28! What does that mean?"
**Emotion:** Excited, possibly surprised
**FundiLink action:** Show APS score prominently; explain what it means; immediate prompt to see matching programmes
**Opportunity:** This is a high-delight moment. Make it celebratory and actionable.

---

### Stage 6: Programme Discovery
**Touchpoint:** Programme matching results
**Thabo's thought:** "Wow, I qualify for more than I thought!"
**Emotion:** Excited, potentially overwhelmed by choice
**FundiLink action:** Filtered, ranked list of matching programmes; strong match / borderline / stretch indicators; save to tracker CTA
**Opportunity:** Reduce overwhelm with smart defaults and filters. Help Thabo shortlist.

---

### Stage 7: Document Preparation
**Touchpoint:** Document checklist for saved applications
**Thabo's thought:** "I need a certified ID copy. Where do I get it certified?"
**Emotion:** Slightly anxious, practical
**FundiLink action:** Clear document checklist; guidance on how to get documents certified; upload interface; status indicators
**Opportunity:** Add simple "how to" guides for first-time applicants.

---

### Stage 8: Application Tracking
**Touchpoint:** Application tracker
**Thabo's thought:** "I've submitted to three places. Now I wait."
**Emotion:** Relieved, anticipatory
**FundiLink action:** Clear status per application; deadline tracker; notification settings
**Opportunity:** Keep Thabo engaged and informed so he doesn't fall off.

---

### Stage 9: Notification and Follow-Up
**Touchpoint:** Email / in-app notification — deadline in 7 days
**Thabo's thought:** "Oh, I forgot about this deadline. Good thing I got reminded."
**Emotion:** Grateful, then action-oriented
**FundiLink action:** Timely, clear deadline notifications; link back to the application
**Opportunity:** Make notifications actionable — deep link to the relevant action.

---

### Stage 10: Outcome
**Touchpoint:** Acceptance (or rejection) received via official portal
**Thabo's thought:** "I got in!" (or "I was rejected — what now?")
**Emotion:** Elated or disappointed
**FundiLink action:** Update application status; celebrate accepted outcomes; guide on next steps; redirect to alternatives if rejected
**Opportunity:** Be there at the outcome moment. Celebrate the wins. Support through the setbacks.

---

## Emotional Journey Summary

```
Awareness → Hopeful/Sceptical
Register   → Slightly Anxious → Reassured
Profile    → Slightly Overwhelmed → Guided
APS Result → High Delight (key moment!)
Matching   → Excited → Risk of Overwhelm
Documents  → Practical/Anxious → Guided
Tracking   → Relieved
Waiting    → Anticipatory → Risk of Drop-Off
Outcome    → High Emotion (celebrate or support)
```

---

## Key Moments That Matter (Design Priority)
1. The APS result reveal — make it a celebration
2. First programme match — make it feel personalised
3. Document checklist — reduce anxiety with clear guidance
4. Deadline notification — must be timely and actionable
5. Acceptance result — capture the moment
