# 17 — School Pilot Proposal

## Title
FundiLink — School Pilot Programme Proposal

## Purpose
Define the structure, requirements, and value proposition for the FundiLink school pilot programme.

---

## Pilot Goal
Validate the FundiLink School Dashboard with real guidance counsellors and Grade 12 learners, gather feedback, and demonstrate value before scaling the school channel.

---

## What the Pilot Offers Schools

### For Learners
- Free access to FundiLink SmartApply
- APS calculator and programme matching
- Document vault and application tracker
- Deadline notifications
- One profile for all applications

### For Guidance Counsellors
- School dashboard showing all Grade 12 learner progress
- See who has completed their profile, uploaded documents, started applications
- Identify learners who need support without manual follow-up
- Replace manual spreadsheets with real-time visibility

### For School Management
- Aggregate data on how many learners are applying to higher education
- Evidence of counsellor support activity
- Useful for reporting to school governing bodies

---

## Pilot Requirements from Schools

| Requirement | Details |
|---|---|
| Grade 12 learner participation | Minimum 20 learners willing to register |
| School admin / counsellor account | One designated school admin |
| Feedback participation | School agrees to provide structured feedback after 8 weeks |
| Communication support | School sends pilot intro to learners and parents |

---

## Pilot Structure

### Week 1–2: Onboarding
- School admin account created
- Counsellor orientation (30-min walkthrough)
- Learner registration campaign (in class or via WhatsApp group)
- Learners create profiles and enter NSC results

### Week 3–6: Active Use
- Learners use FundiLink for programme matching and document organisation
- Counsellor uses dashboard to track and support learners
- FundiLink team provides support via WhatsApp group or email

### Week 7–8: Feedback and Review
- Structured feedback session with counsellor
- Learner survey (short, mobile-friendly)
- Case study collection (with permission)

---

## Pilot Schools Profile (Ideal)

| Criteria | Target |
|---|---|
| Number of pilot schools | 3–5 |
| Location | Mix: rural (Limpopo/Eastern Cape/KZN) + urban/township |
| School type | Public school |
| Grade 12 size | 30–150 learners |
| Connectivity | At least counsellor has internet access |
| Counsellor motivation | Open to using new tools |

---

## What FundiLink Provides

- Free access to the platform for all learners and school admin
- Onboarding and training for the school admin / counsellor
- Ongoing support during the pilot
- Summary report for the school at the end of the pilot
- School listed as a founding pilot partner (with permission)

---

## Success Metrics for the Pilot

| Metric | Target |
|---|---|
| Learner registration rate | >60% of Grade 12 class |
| Profile completion rate | >50% of registered learners |
| Document uploads | >40% of registered learners upload at least 1 document |
| Counsellor engagement | Counsellor logs in at least weekly |
| NPS score (learners) | >50 |
| NPS score (counsellors) | >60 |
